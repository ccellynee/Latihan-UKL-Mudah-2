Bagian 1: 

        System.out.println("----- Menentukan bilangan ganjil atau genap -----"); 
        System.out.println("Masukkan bilangan anda: ");
        int angka =input.nextInt();
        
Pada bagian tesebut program meminta pengguna untuk menginput bilangan (yang akan ditentukan ganjil atau genap).



Bagian 2:

        if (angka % 2 == 0) {
            System.out.println(angka + " adalah bilangan genap");
        } else {
            System.out.println(angka + " adalah bilangan ganjil");
        }

Program menentukan bilangan yang dimasukkan ganjil atau genap. Dengan kondisi jika bilangan yang dimasukan ketika dibagi 2 sisa 0 atau habis, maka bilangan tersebut termasuk ke bilangan genap. Dan ketika ada bilangan yang tersisa atau tidak habis dibagi 2, maka bilangan termasuk ke bilangan ganjil.
